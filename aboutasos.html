About ArchSawOS

ASOS (ArchSawOS) is a Arch Linux based distro that uses the XFCE4 display manager and the LightDM login manager.

The installer for ASOS is calamares which simplifies the user experience when installing.

ASOS includes a sleek forest green color scheme.

ASOS does include Firefox out of box. (really isnt a feature point but cmon, its something)
